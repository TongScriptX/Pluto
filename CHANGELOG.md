## 📋 版本说明

### 版本号规则
- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 发布周期
- **主版本**: 根据需要发布
- **次版本**: 每月发布一次
- **修订版**: 根据需要发布

### 支持政策
- **当前版本**: 完全支持，包含新功能和修复
- **前一个主版本**: 仅安全修复和严重 bug 修复
- **更早版本**: 不再维护
